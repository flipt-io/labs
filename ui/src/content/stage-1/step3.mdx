# Creating the Flag

In the `docker-compose.yml` file that you have already run, we've also spun up a Flipt server backed by a SQLite database.

The Flipt server is running on port `8080`. Click the link [http://localhost:8080/](http://localhost:8080/) to open the Flipt UI.

![Flipt UI](/images/flipt-ui.png)

Next, click on the `New Flag` button to create a new flag.

![New Flag](/images/new-flag.png)

Enter the following details:

- Name: `Chat Enabled`
- Key: `chat-enabled`
- Description: `Enable chat for all users`

Finally, click the 'Enabled' toggle to enable the flag and then click the 'Create' button.

Click 'Next' to continue.
