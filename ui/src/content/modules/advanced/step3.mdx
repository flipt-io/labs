# Create the Flag and Rule

We're finally ready to create the `chat-admin` flag and rule which will enable the admin functionality feature for internal users.

## Creating the Flag

Click the link [http://localhost:8080/](http://localhost:8080/) to open the Flipt UI.

Next, click on the 'New Flag' button to create a new flag.

![New Flag](/images/advanced/new-flag.png)

Enter the following details:

- Name: `Chat Admin`
- Key: `chat-admin`
- Description: `Enable admin mode for internal users`

Click the 'Enabled' toggle to enable the flag and then click the 'Create' button.

## Creating the Rule

Next, click on the 'Evaluation' tab and then click on the 'New Rule' button.

![New Rule](/images/advanced/new-rule.png)

Enter the following details:

- Segment: `admins`
- Type: `Single Variant`

**Note**: We won't actually be using variants in this example, but we need to select a variant type in order to create the rule. The concepts guide in our documentation has more information on [variants](https://www.flipt.io/docs/concepts#variants).

Click the 'Create' button.

Click 'Next' to continue.
