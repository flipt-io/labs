# Creating the Flag

In this section, we will create a new flag in Flipt.

## Setup

<Well>
**Required**: In the `chatbot` folder, run the following command to start the Flipt server and our other backend services required for this tutorial:

```bash
docker-compose -f docker-compose.basic.yml up
```

</Well>

Once initialized, the Flipt server will be running on port `8080`.

## Creating the Flag

1. Click the link [http://localhost:8080/](http://localhost:8080/) to open the Flipt UI

![Flags](/images/basic/flags.png)

2. Next, click on the 'New Flag' button to create a new flag

![New Flag](/images/basic/new-flag.png)

3. Enter the following details:

```
- Name: `Chat Enabled`
- Key: `chat-enabled`
- Description: `Enable chat for all users`
```

4. Click 'Create'

Click 'Next' to continue.
